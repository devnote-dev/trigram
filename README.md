# trigram

Based on [kkdai/trigram](https://github.com/kkdai/trigram).

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     trigram:
       github: devnote-dev/trigram
   ```

2. Run `shards install`
